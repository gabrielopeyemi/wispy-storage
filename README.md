# Wispy storage

### Installation of wispy

npm i --save wispy 


### Using wispy storage:

To add data to a store:
addToWispy('details': {name: 'gabe', age: 17});

To get data from store
getFromWispy('details');

To delete a store
deleteFromWispy('details');

To delete all stores
cleanWispy();
